=======================
Environment
=======================

In this section, we set up a development environment for CRConnect.

.. Note::

    To set up your environment, you need Python 3, node.js, Angular, and Docker installed.

    In particular, you must be able to run

    .. code-block::

        $ pipenv

    .. code-block::

        $ npm

    .. code-block::

        $ docker-compose
